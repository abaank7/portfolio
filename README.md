# Abaan Koul | Developer Portfolio 🚀

A highly interactive, performance-optimized personal portfolio built with React, TypeScript, and Vite. Designed with a custom **Autonomous ROS HUD** aesthetic to reflect my focus on bridging scalable full-stack web architecture with intelligent, autonomous machine learning systems.


## ⚡ Features

* **Custom UI Architecture:** A highly unique, hardware-inspired "ROS HUD" theme built entirely from scratch using Tailwind CSS, featuring terminal-style typography, clipping paths, and neon phosphor glows.
* **Modular Component Design:** Clean, scalable React architecture with fully decoupled components (`Hero`, `Experience`, `Projects`, `Skills`, etc.) driven by dynamic JSON data.
* **Scrollspy Navigation:** A custom IntersectionObserver hook that seamlessly tracks the user's viewport and updates the navigation state with smooth animations.
* **Type-Safe:** Written entirely in TypeScript for robust, error-free component rendering and prop validation.
* **Dark-Mode Native:** Engineered with deep, high-contrast colors specifically optimized for developer environments and readability.

## 🛠️ Tech Stack

* **Framework:** React 18
* **Build Tool:** Vite (configured with React Compiler/SWC for lightning-fast HMR)
* **Language:** TypeScript
* **Styling:** Tailwind CSS (v4)
* **Typography:** Google Fonts (Fira Code)
* **Deployment:** Vercel / Netlify *(Update based on your host)*
